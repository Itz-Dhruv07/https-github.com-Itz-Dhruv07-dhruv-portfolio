# Dhruv's Developer Portfolio

A sleek, Discord-inspired developer portfolio built with React, Tailwind CSS, and Motion.

## 🚀 Features
- **Discord UI/UX**: Authentic dark theme, pills, and card layouts.
- **Dynamic Projects**: Searchable repository list with animated filtering.
- **Skills Showcase**: Categorized skills including 10+ programming languages.
- **Responsive**: Fully optimized for mobile and desktop.
- **Auto-Deploy**: Pre-configured with GitHub Actions for GitHub Pages.

## 🛠️ Tech Stack
- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS 4.0
- **Animations**: Motion (Framer Motion)
- **Icons**: Lucide React

## 📦 Getting Started

1. **Clone the repo**:
   ```bash
   git clone https://github.com/Itz-Dhruv07/dhruv-portfolio.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run development server**:
   ```bash
   npm run dev
   ```
